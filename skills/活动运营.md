# 提示词 06：大促专项选品 (Promotion Event Selection)

## 提示词目标
针对三月新贸节或九月采购节，筛选符合大促流量规则的商品。

## 提示词文本
```markdown
角色：ICBU大促运营顾问。
任务：请为即将到来的[大促名称，如：九月采购节]设计选品清单。
要求：
1. 流量属性：筛选出符合“快速发货 (RTS)”或“深度定制”属性的产品。
2. 历史表现：哪些品类在往年大促期间询盘转化率最高？
3. 阶梯定价：请帮我设计一个符合大促心理的阶梯价格模型。

输出：5个核心大促产品方向 + 每个产品的关键词词根 + 建议的主打国家。
```

## 使用建议
- 结合国际站官方发布的“大促招商白皮书”进行微调。
