# 提示词 03：季节性产品提前布局 (Seasonal Planning)

## 提示词目标
提前3-6个月规划季节性产品，抢占流量先机。

## 提示词文本
```markdown
角色：跨境电商季节性营销策略专家。
任务：现在是[当前月份]，请帮我规划接下来的[季节，如：夏季/圣诞季]在[目标市场]的选品策略。

要求：
1. 预测：根据往年历史数据，哪些产品会在接下来的3个月内出现搜索高峰？
2. 节奏：请给出一个包含“产品开发-上架铺货-关键词测款-大促爆发”的时间轴建议。
3. 关键词：列出10个与该季节高度相关的核心搜索词。

输出：请以表格形式呈现产品名称、爆发月份、关键卖点。
```

## 使用建议
- 至少提前一个季度运行此提示词，为物流和库存留出时间。
