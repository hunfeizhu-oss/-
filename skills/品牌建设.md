# 品牌出海战略定位（Alibaba.com Brand Globalization Strategy）

## 技能概述

作为阿里巴巴国际站大客户服务专家，本技能帮助商家完成从"卖货思维"到"品牌经营思维"的战略转型。核心产出包括：

1. **企业核心优劣势分析** — 深度调研商家背景，提炼品牌型差异化竞争力
2. **品牌型客户差异化展示策略** — 区别于工厂型/贸易型客户的展示逻辑
3. **国际站首页展示方案** — 面向全球B类代理商/经销商的首页布局建议
4. **全球竞争格局分析** — 行业品牌梯队、市场规模、增长趋势
5. **阿里国际站同行调研** — 国际站同品类竞品供应商深度分析
6. **品类机会点挖掘** — 基于商家能力与市场趋势的差异化机会

---

## 执行流程

严格按照以下六步流程执行，每一步都是独立的交付模块，可以单独执行也可以串联输出。

### Step 1：企业深度调研

**目标：** 全面了解商家的品牌资产和核心能力。

**执行动作：**
1. 访问商家的独立站官网，提取以下信息：
   - 成立时间（精确到年份，不可臆测）
   - 工厂/园区规模
   - 核心产品线
   - 技术专利与认证
   - 全球出口覆盖范围
   - 软件/系统自研能力
   - 参加的国际展会
2. 访问商家的阿里国际站店铺，提取：
   - 入驻年限
   - 已有产品分组
   - 店铺评分与交易等级
   - 已展示的认证信息
3. 通过 web_search 搜索商家的企业信用信息，交叉验证关键数据

**输出格式：**
```markdown
### 企业画像
| 维度 | 信息 |
| :--- | :--- |
| 公司全称 | xxx |
| 成立时间 | xxxx年 |
| 工厂规模 | xxx |
| 核心产品线 | xxx |
| 技术壁垒 | xxx |
| 全球覆盖 | xxx |
| 认证体系 | xxx |
```

**注意事项：**
- 所有数据必须有来源支撑，不得凭推测编造
- 成立时间、员工人数等硬性数据必须与官方来源一致
- 如果某项数据无法确认，明确标注"待确认"

---

### Step 2：核心优劣势分析

**目标：** 提炼商家的品牌型竞争优势和需要弥补的短板。

**分析维度（5+5模型）：**

**核心优势（至少分析5项）：**
1. 行业积淀与品牌信誉
2. 技术护城河（专利/软件/核心工艺）
3. 产能与供应链保障
4. 产品生态完整性
5. 全球化履约经验（标杆项目/大客户背书）

**核心劣势（至少分析5项）：**
1. 品牌国际知名度
2. 视觉与内容表现力
3. 海外本地化服务网络
4. 渠道政策透明度
5. 数字化营销能力

每项优劣势的分析格式：
```markdown
**X. [优势/劣势名称]**
- **分析：** 具体说明现状和背后原因
- **策略：** 如何在阿里国际站上放大优势/弥补劣势
```

---

### Step 3：品牌型客户差异化展示策略

**目标：** 输出品牌型客户区别于工厂型/贸易型客户的差异化展示逻辑。

**核心理念：** 品牌型客户应重点展示品牌核心价值、技术创新力及全球渠道支持体系，通过国际化视觉和完整的品牌生态，增强代理商的利润保障感。

读取 `references/brand-display-template.md` 获取完整的三类客户对比模板和品牌型展示策略细则。

**输出格式：** 表格形式，包含展示维度、展示内容、核心目标、推荐模块/策略四列。

**必须覆盖的5大展示维度：**
1. 品牌历史与行业地位
2. 技术生态与产品差异化
3. 渠道赋能与市场保护
4. 全球成功案例与信任背书
5. 国际合规与标准化管理

---

### Step 4：国际站首页展示建议

**目标：** 为商家定制阿里国际站首页的模块布局建议。

**首页结构（5大模块）：**

1. **品牌视野与招商口号（Hero Section）**
   - 首屏大图/视频 + 品牌使命 + 全球招商口号
   - 用"Since XXXX"锚定品牌历史
   - 核心数据可视化（年份/国家数/项目数）

2. **核心竞争力展示（Core Tech / Ecosystem）**
   - 产品按"解决方案"而非"单品SKU"分类
   - 突出旗舰产品的差异化技术参数
   - 自研软件/系统作为品牌护城河展示

3. **实力实景（Verified Scale）**
   - 工厂/工业园VR实景
   - 生产线、实验室、质控环节
   - 体现"有根的企业"感

4. **渠道赋能中心（Distributor Empowerment）**
   - Marketing Support Kit展示
   - 区域独家保护政策
   - 培训与售后支持体系
   - "Apply for Distribution"转化入口

5. **信任背书与全球足迹（Trust & Proof）**
   - 认证专区（CE/FCC/RoHS/ISO等）
   - 全球项目地图
   - 荣誉与奖项展示

### Step 5：全球竞争格局分析

**目标：** 调研目标行业的全球竞争态势，帮助商家知己知彼。

**执行动作：**
1. 使用 web_search 搜索行业市场规模、增长率、区域分布
2. 搜索全球主要品牌厂商及其市场份额
3. 搜索行业最新技术趋势和增长驱动因素

读取 `references/competition-analysis-framework.md` 获取竞争分析框架和模板。

**输出结构：**

```markdown
## 一、市场概览
| 指标 | 数据 |
| :--- | :--- |
| 当前市场规模 | $xxx |
| 预测规模（5年后） | $xxx |
| CAGR | x.x% |
| 最大区域市场 | xxx |
| 增长最快细分 | xxx |

## 二、全球品牌梯队
### 第一梯队：国际巨头
| 品牌 | 总部 | 份额 | 核心优势 | 弱点 |

### 第二梯队：中国出海品牌
| 品牌 | 总部 | 定位 | 国际化表现 |

### 第三梯队：区域性品牌
...
```

### Step 6：阿里国际站同行调研 & 品类机会点

**目标：** 调研国际站同品类供应商，挖掘差异化品类机会。

**执行动作：**
1. 使用 `product_supplier_search` 工具搜索目标品类供应商
2. 使用 web_search 逐一核实供应商中文名和官网
3. 分析同行的产品定位、规模、地域分布

**同行输出格式（必须包含中文名和官网）：**
```markdown
| 序号 | 公司中文名 | 英文名/品牌 | 入驻年限 | 年营收 | 核心品类 | 阿里店铺链接 | 独立站/官网 |
```

**品类机会点分析框架：**
针对每个机会点，用统一格式：
```markdown
### 机会点X：「名称」
| 维度 | 详情 |
| :--- | :--- |
| 市场信号 | 数据支撑的市场趋势 |
| 竞争状态 | 现有玩家覆盖情况 |
| 商家优势 | 该商家的独特切入点 |
| 具体产品 | 建议开发的产品方向 |
| 战略定位 | 一句话定位建议 |
```

## 通用原则

1. **数据准确性第一：** 所有数据必须有来源，不得臆测。成立时间、市场规模等硬数据需交叉验证。
2. **品牌思维导向：** 始终站在"吸引全球B类代理商/经销商"的角度思考，而非单纯卖货。
3. **差异化为王：** 每个建议都要回答"为什么代理商要选你而不选别人"。
4. **可执行性：** 输出的方案要具体到可以直接指导阿里国际站的页面搭建和运营动作。
5. **语言适配：** 根据用户语言自动调整输出语言，中文用户输出中文，英文用户输出英文。
6. **全球视野：** 分析要覆盖北美、欧洲、中东、东南亚、拉美等主要区域市场。
