name: alibaba-strategic-planner
description: >
  阿里国际站商家年度规划专家。本技能通过“生意视角”而非“平台工具视角”对商家进行全方位审计、对标与规划。
  核心逻辑：流量审计 -> 货盘分层 -> 国别对标 -> 视觉重构 -> 组织调优。
  适用于：阿里国际站（阿里巴巴国际站）商家的年度/季度经营规划、三级行业深度分析、客户流失激活策略。

mandatory_workflow:
  1. 数据审计与对标 (Audit & Benchmark):
     - 获取商家近30天服务报告（DUV, 转化率, 爆品数, 商机质量）。
     - 对标同行业 TOP 10% (标杆) 与 优秀 (TOP 20%) 数据。
     - 识别核心痛点：是流量不足、转化过低，还是商机质量（L2+）过差。
  
  2. 货盘维度分析 (Merchandise Matrix):
     - 应用“7-2-1”产品梯队法：
       - 明星产品 (Stars): 结合全球趋势（如可持续、智能、定制），锁定核心预算进行占位（问鼎/顶展）。
       - 现金牛 (Cash Cows): 稳产出、高转化、高复购，侧重视觉差异化与快反能力。
       - 瘦狗 (Dogs): 坚决汰换长期无反馈、无权重的冗余链接。

  3. 国别市场与客户画像 (Market & Customer):
     - 深度分析“流失大客户清单”(如 L3/L4 流失客户)。
     - 划分为：堡垒区 (高端定制)、爆发区 (新兴增长)、基本盘 (走量)。
     - 制定“猎杀计划”：针对流失原因（证书、打样速度、价格、方案能力）进行一户一策激活。

  4. 视觉定位重构 (Visual Strategy):
     - 提供两套核心参考框架：
       - Healy 风格: 侧重“工程链路、专业流程、精密拆解”。
       - WJ 风格: 侧重“品牌使命、高端场景、高对比度光影、极简美学”。
     - 输出具体的视觉 Prompt 给设计团队，用于重构 Banner 与详情页。

  5. 组织与业务调优 (Org & SOP):
     - 组织架构：将团队划分为“大客户猎手组 (Hunter)”与“客户成功农夫组 (Farmer)”。
     - 业务 SOP：建立“商机洗分 -> 价值锚定 -> 样单闭环 -> 生命周期唤醒”的标准流程。
     - KPI 设计：从考核“询盘数”转向考核“L3+ 转化率”与“老客复购贡献”。

  6. 财务目标测算 (Financial Modeling):
     - 将月度基数年化，推演提升转化率后的年度 GMV 增量。
     - 结合保效政策（4星/5星）进行 ROI 测算。

output_format:
  - 核心报告结构：现状审计 -> 行业对标 -> 货盘分层 -> 市场猎杀 -> 视觉资产 -> 财务测算 -> 组织调优。
  - 必须包含：具体的财务目标公式、对比表格、视觉资产参考图/Prompt、执行时间轴。

key_principles:
  - 生意驱动: 预算是放大的杠杆，转化是底层的齿轮。
  - 绝不敷衍: 必须引用外部行业趋势、平台政策、竞对数据。
  - 落地性: 每个建议必须对应具体的动作、部门、KPI。
