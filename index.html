Accio Work 功能介绍

国际站 AI 智能工作台全面解析

平台概览 必读

AccioWork 是阿里巴巴国际站为跨境卖家量身打造的 AI 工作台。通过内置的大语言模型能力，卖家可以在平台内直接完成商品描述生成、营销文案撰写、智能体搭建与 Skill 技能调用等工作，无需切换多个工具。

🤖

智能体（Agent）

自定义 AI 角色，设定专属工作场景与知识背景，让 AI 成为专业领域专家。

⚙️

技能（Skill）

封装标准化任务流程，一键调用固定提示词模板，统一团队输出规范。

✍️

提示词工程

通过优质提示词精准控制 AI 输出，覆盖产品、营销、客服全场景。

📦

技能包共享

国际站官方提供分阶段技能包，覆盖店铺全生命周期运营需求。

🌐

多语言支持

支持英、西、葡、阿等主流国际语言，无缝覆盖全球买家市场。

📊

数据洞察

结合平台数据生成运营分析报告，辅助卖家做出科学决策。

工作流程
1

登录 AccioWork 工作台

使用国际站卖家账号登录，进入 AccioWork 主界面。

2

选择或创建 Agent

从官方模板库选择预设智能体，或根据业务需求自定义搭建。

3

调用 Skill 技能

在对话界面选择对应的 Skill，输入变量参数后一键生成内容。

4

优化与迭代

根据输出效果调整提示词或 Skill 配置，持续提升生成质量。

🔗 深度集成国际站数据 ⚡ 无需代码即可使用 🤝 团队协作共享 📐 标准化输出模板 🔒 数据安全隔离 🌍 全球化语言能力

以刚毕业大学生从 0 启动为案例，按“每天可执行动作”设计，不讲空概念，而是完成从产品方向、供应链、店铺资产到客户开发和转化的验证闭环。

Agent 搭建步骤
1

进入 Agent 管理页面

在 AccioWork 主界面点击「我的智能体」→「新建智能体」。

2

设置角色定义（System Prompt）

为 Agent 起名并编写角色介绍，明确其职能定位（如：专业外贸文案专家、产品详情页撰写师）。建议包含：身份定义 + 专业背景 + 工作目标 + 输出规范。

3

上传知识库（可选）

上传产品手册、公司介绍、行业资料等文件，让 Agent 基于专属知识回答问题。

4

挂载 Skill 技能

为 Agent 绑定常用 Skill，使其在对话中可以直接调用标准化任务模板。

5

测试与发布

在预览窗口输入测试问题，检验 Agent 的回答质量，满意后保存并分享给团队。

注意事项 重要
!

角色设定要具体

避免过于笼统的描述，越具体的角色设定，AI 的输出越专业。例如不要写"你是助手"，而应写"你是专注于宠物用品行业的 B2B 外贸文案专家，擅长为北美买家撰写产品详情页"。

!

知识库文件格式要求

支持 PDF、Word、TXT、MD 格式，单文件建议不超过 10MB，文字内容为主，避免纯图片文件。

!

定期迭代优化

根据实际使用中的反馈不断调整 System Prompt，好的 Agent 需要持续打磨。

Skill 创建与使用步骤
1

进入 Skill 管理

在 AccioWork 工作台点击「技能库」→「新建技能」，或从官方技能包导入。

2

编写 Skill 提示词模板

在模板中设置固定的指令内容，并用「{{变量名}}」标记需要用户填写的动态参数，例如 {{产品名称}}、{{目标市场}}。

3

配置变量说明

为每个变量添加说明文字和示例，帮助使用者正确填写输入内容。

4

保存并绑定 Agent

将 Skill 保存后，可将其挂载到对应的 Agent 下，方便在对话中快捷调用。

5

团队共享

将高质量 Skill 分享给团队成员，统一输出标准，提升整体运营效率。
