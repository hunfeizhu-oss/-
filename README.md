# 智启新程 · 战报生成器

这是一个本地运行的战报生成工具。它会读取当前目录中的 PPTX 模板，将表单内容写回模板，并通过 macOS Keynote 或 Windows PowerPoint 自动导出 PNG 图片。

## 使用方法

1. macOS 双击 `start.command`；Windows 双击 `start.bat`。
2. 浏览器会自动打开本地页面，默认地址为 `http://127.0.0.1:8765`。如果端口已占用，工具会自动顺延。
3. 选择模板，上传人物照片并填写缺少的战报信息，点击“生成战报图片”。
4. 在右侧预览并下载 PNG 或 PPTX。
5. 终端窗口中按 `Ctrl+C` 可停止工具。

## 给其他人使用

有两种方式：

1. 独立使用：将整个工具目录发给同事。同事在自己的电脑上双击 `start.command` 或 `start.bat`。
2. 局域网共享：在一台已安装 Keynote 或 PowerPoint 的电脑上双击 `start-share.command` 或 `start-share.bat`。终端会显示类似 `http://192.168.1.20:8765` 的共享地址，同一 Wi-Fi 或内网中的同事可直接用浏览器访问。

共享模式仅建议在可信任的公司内网中使用。所有生成任务会自动排队，由开启共享的电脑统一完成转图。关闭启动窗口或按 `Ctrl+C` 即可停止共享。

## 系统要求

- macOS：安装 Keynote。首次自动导出时，系统可能会询问是否允许 Python 控制 Keynote，请选择允许。
- Windows：安装 Python 3 和桌面版 Microsoft PowerPoint。安装 Python 时请勾选 `Add Python to PATH`。

人物照片建议使用透明背景 PNG，也支持 JPG。macOS 还可尝试 WebP、HEIC 等系统支持的格式。单张请控制在 12MB 以内。上传后会按照模板图片框等比缩放并居中裁切，不会拉伸变形。未上传照片时，工具会保留模板中的原图片。A200 模板支持左右两张人物照片；Q1 横版模板支持额外上传 5 张战报图片。

填写后的动态文字会尽量沿用模板原生风格。产品与项目标题仅增加克制的细描边和轻阴影，金额保留模板渐变并做轻量强化；姓名、公司、团队、主管和经验分享保持模板本身的视觉体系。较长内容会自动缩小，尽量避免溢出。

生成后的海报底部会自动加入固定文案：`星芽铁军 战无不胜 攻无不克`。

## Windows 启动排查

1. 将整个工具目录完整解压后再运行，不要直接在压缩包预览窗口中双击。
2. 正常启动请双击 `start.bat`；局域网共享请双击 `start-share.bat`。
3. 如果无法启动，双击 `windows-diagnose.bat`。诊断窗口会检查 Python、模板和 PowerShell。
4. 启动器会在工具目录生成 `windows-start.log`。如果仍然失败，请提供窗口中的报错和该日志文件。

Windows 需要 Python 3.9 或更高版本。新版启动器会自动尝试 `py -3`、`python`、`python3` 以及常见 Python 安装目录；失败时窗口会停留显示原因，不会直接闪退。

## 添加模板

将新的 `.pptx` 文件放进本目录，或直接覆盖同名模板，然后点击页面中的“更新最新模板”。无需关闭或重启工具。工具会重新扫描目录，并自动识别第一页中可见的文本框，生成通用编辑表单。内置的 5 份模板已经配置了更易读的中文字段名；如果新版模板重建了文本框，新增内容也会自动作为通用字段显示。

## 文件位置

每次生成的文件也会保存在 `generated/` 目录。图片导出失败时，PPTX 仍会保留，可使用 Keynote 或 PowerPoint 手动导出。
